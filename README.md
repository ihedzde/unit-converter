# Unit Converter
## Unit converter on SwiftUI

This app supports converting  units in following categories:
- Volume
  - Milliliters
  - Liters
  - Cups
  - Pints
  - Gallons
- Temperature
  - Celsius
  - Fahrenheit
  - Kelvin
- Time
  - Second
  - Minute
  - Hour
  - Day
- Length
  - Kilometers
  - Feet
  - Yard
  - Mile
